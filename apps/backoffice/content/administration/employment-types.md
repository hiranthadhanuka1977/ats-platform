## About this list

Employment types classify the working arrangement for a posting.

- **Slug** — Short stable code (for example `full-time`) used in URLs and reporting.
- **Sort order** — Display order in posting forms and filters.

Avoid duplicates; each type should have a clear, distinct meaning for recruiters and candidates.
