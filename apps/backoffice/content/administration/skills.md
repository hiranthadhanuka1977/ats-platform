## About this list

Skills are a shared vocabulary across postings. Recruiters pick from this list when defining role requirements.

- **Name** — Must be unique; use consistent spelling and casing for reporting.
- Deleting a skill may be blocked while it is still linked to postings—remove links from postings first if required.
